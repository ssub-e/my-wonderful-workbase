# Value Proposition (가치 제안) 및 Feature Map

수요 예측 AI SaaS의 핵심 가치 제안과 MVP 기능 우선순위(MoSCoW) 맵입니다.

## 1. 가치 제안 (Value Proposition)
1. **MD 결재 방어 PDF 리포트**: 기상/트렌드/매출 변수를 융합하여 상사를 설득해줄 'XAI 해설 엔진' 탑재 리포트. 기존 수기 엑셀(4시간) 작업을 10분으로 단축시키고 결재 반려율을 0으로 만듦.
2. **스마트 센터장 인건비 옵티마이저**: 1-Click 인증으로 화주사 플랫폼 API를 연동하여 내일 출근할 최적 알바 인원수를 역산/제시. 잉여 알바 인건비(매일 80만 원) 누수 차단.

## 2. MVP 핵심 기능 (MoSCoW - Must)
- **F1**: [결재 방어용] 날씨/트렌드 융합 XAI 리포트 추출기
- **F2**: [1-Click] 이커머스 쇼핑몰 API 원클릭 통합 모듈
- **F3**: [인건비 방패] 알림톡 기반 일일 적정 알바 역산 대시보드

*참고: 콜드체인 실시간 노선 재설정기(F4)는 제품 코어 방향을 해치므로 영구 제외(Won't) 결정됨.*
