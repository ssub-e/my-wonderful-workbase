## 🧪 Acceptance Criteria (BDD/GWT)
Scenario 1: 브랜드 컬러 변경 및 PDF 반영
- Given: 화주사 A가 브랜드 컬러를 'Navy (#000080)' 로 설정함
- When: PDF 리포트를 생성함
- Then: 생성된 PDF의 타이틀 바와 표 헤더 색상이 #000080으로 자동 채색되어 출력된다.

Scenario 2: 로고 이미지 Presigned URL 처리
- Given: S3에 업로드된 테넌트 로고가 존재함
- When: 리포트 생성 엔진이 실행됨
- Then: `FILE-001` 어댑터를 통해 유효한 Presigned URL을 획득하여 PDF 렌더러가 외부 경로의 로고를 정상적으로 박아 넣는다.
